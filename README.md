# IPsec 配置大师 🎓

一款用于学习 IPsec 配置知识的交互式学习工具。

## 📖 项目简介

IPsec Configuration Master 是一个基于 PyQt5 的桌面应用程序，专为学习 IPsec（Internet Protocol Security）技术而设计。通过知识学习和实践挑战相结合的方式，帮助用户掌握 IPsec 的核心概念和配置技能。

## ✨ 功能特性

### 📚 知识学习
- IPsec 概述与核心组件
- IKE 协议详解（Phase 1/Phase 2）
- 加密与认证算法对比
- NAT 穿越技术
- 高可用与故障切换
- DPD 与隧道维护

### 🎯 练习挑战
5 个不同难度的配置关卡：
1. **IKE 基础配置** - 入门级
2. **IPsec 策略配置** - 基础级
3. **NAT 穿越场景** - 进阶级
4. **双机热备 HA** - 高级
5. **复杂多对多 VPN** - 专家级

### 🏆 成就系统
- 学习进度追踪
- 成就解锁机制
- 分数统计与排行

## 🚀 快速开始

### 环境要求
- Python 3.8+
- PyQt5

### 安装依赖

```bash
pip install PyQt5
```

### 运行程序

```bash
python ipsec_master.py
```

### 运行测试

```bash
python -m unittest test_ipsec_master -v
```

## 🎮 使用说明

1. **知识学习**：点击主菜单的"知识学习"按钮，浏览 IPsec 相关知识模块
2. **练习挑战**：选择不同难度的关卡，根据场景描述完成配置
3. **提交答案**：填写配置参数后点击"提交答案"查看得分
4. **获取帮助**：可使用"显示提示"或"显示参考"按钮获取帮助
5. **查看成就**：在成就系统中查看学习进度和已解锁成就
